cs494 final project README
To run the server you will need to install Node.js and
npm (This should come with Node.js). Once these are
downloaded you need to use npm to install Socket.io and express.
the commands for these are:
 npm install socket.io
 npm install express
for the client type http://localhost:8080/chat.html in you browser with
the server running and the client should start up. Chrome and Firefox have been tested.
